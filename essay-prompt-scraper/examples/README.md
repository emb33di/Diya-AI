# Essay Prompt Scraper Examples

This directory contains various examples demonstrating how to use the essay prompt scraper.

## Files

- `basic-examples.js` - Basic usage examples
- `advanced-scraper.js` - Advanced scraping techniques
- `custom-extractors.js` - Custom extraction logic examples

## Running Examples

### Basic Examples
```bash
cd essay-prompt-scraper
npm install
node examples/basic-examples.js
```

### Advanced Scraper
```bash
node examples/advanced-scraper.js
```

## Example Types

### 1. Basic Scraping
Simple scraping of configured university websites.

### 2. Data Processing
Scraping with automatic data cleaning and analysis.

### 3. Specific Universities
Scraping only selected universities from the configuration.

### 4. Custom Configuration
Using custom scraping settings and targets.

### 5. Error Handling
Demonstrating error handling and retry logic.

### 6. Advanced Techniques
- Dynamic content scraping
- JavaScript-heavy websites
- Multiple application systems (Common App, Coalition, UC)
- Custom extraction logic

## Customization

You can modify the examples to:
- Add new university websites
- Implement custom extraction logic
- Change output formats
- Adjust scraping parameters
- Add new data processing steps

## Output

Examples save results to the `data/` directory with timestamps for easy tracking.
